# 使用说明

## 一、服务器部署

### 1. 编译安装

```bash
cd my_backup
mkdir -p build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
sudo make install
```

### 2. 修改配置（可选）

```bash
sudo vim /etc/backup-server.conf
```

默认内容：

```ini
[server]
port = 8848
storage_path = /var/lib/backup-server/data
log_file = /var/log/backup-server.log
max_connections = 100
```

### 3. 启动服务器

```bash
sudo systemctl start backup-server
```

查看状态：

```bash
systemctl status backup-server
journalctl -u backup-server -f
```

### 4. 开放端口

在云服务器安全组中放行 TCP `8848` 端口（本地测试可跳过）。

---

## 二、客户端使用（GUI）

### 1. 安装 Qt6（仅首次）

```bash
sudo apt install qt6-base-dev
```

### 2. 编译

```bash
cd my_backup
mkdir -p build && cd build
cmake ..
make -j$(nproc)
```

### 3. 启动

```bash
./backup-gui
```

### 4. 界面操作

GUI 包含 5 个选项卡：

| 选项卡 | 功能 |
|--------|------|
| 本地备份 | 选择目录 → 打包/压缩/加密 → 生成 `.bak` |
| 本地还原 | 选择 `.bak` → 还原到目标目录 |
| 远程备份 | 填写服务器地址和账号 → 备份到服务器 |
| 远程还原 | 从服务器下载备份 → 还原到本地 |
| 远程列表 | 查看服务器上的备份历史 |
